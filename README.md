# Ancient Gemstone Quest

Ancient Gemstone Quest is a Vite + React experience centered on an interactive royal bracelet and the historical treasures of Sri Lanka.

## Development

Requirements:

- Node.js
- npm

Run locally:

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev` starts the Vite dev server.
- `npm run build` creates a production build.
- `npm run preview` serves the production build locally.
- `npm run test` runs the Vitest suite.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Three Fiber
